# Git Multiline Commit Message Generator

A simple web tool to convert your plain text commit message into the correct multiline command for CMD, PowerShell, or Bash/Git Bash.

## ✨ Features
- Paste your commit message (subject + body)
- Instantly get the command for your shell (CMD, PowerShell, Bash)
- One-click copy to clipboard
- Responsive and clean UI

## 🛠 Usage
1. Write or paste your commit message in the textarea.
2. Select your terminal/shell.
3. Copy the generated command and run it in your terminal.

Just push the `index.html` to any static host (GitHub Pages, Netlify, Vercel).
